# Focus

Read the following files and synthesize a focused recommendation:

1. `specs/ops/PRIORITIES.md` — active workstreams and their status
2. `specs/ops/SCORECARD.md` — current metrics
3. `specs/ops/proof-points.md` — stacked causes and unchecked actions

## Output

Produce a short recommendation:
- **Current state**: 1-2 sentences on where things stand
- **Recommended focus**: The single most impactful thing to work on next
- **Why**: One sentence on why this over alternatives
- **First step**: The concrete action to start with
